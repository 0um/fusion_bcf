# fusion_xml
Command line tool to merge two xml files

**Only Windows Platform!**

Build
````
cmake -G "Visual Studio 15 2017"
```

Run
```
fusion_xml.exe -id filter inputA inputB <optional OutputC>
<optional id filer> -id <filter>

```
